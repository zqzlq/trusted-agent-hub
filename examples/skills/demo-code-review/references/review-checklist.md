# Review Checklist

## Correctness
- [ ] 是否有未处理的 null/undefined？
- [ ] 边界条件是否正确？
- [ ] 异常处理是否完整？
- [ ] 类型转换是否安全？

## Security
- [ ] 是否有 SQL/命令注入风险？
- [ ] 用户输入是否经过验证和转义？
- [ ] 敏感信息是否硬编码？
- [ ] 权限检查是否到位？

## Performance
- [ ] 是否存在 N+1 查询？
- [ ] 大文件/大数据是否分页处理？
- [ ] 是否有不必要的重复计算？
- [ ] 异步操作是否正确使用？

## Maintainability
- [ ] 命名是否清晰？
- [ ] 函数是否过长（>50 行）？
- [ ] 是否有重复代码？
- [ ] 关键逻辑是否有注释？
